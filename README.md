# ❄️ 工业冷库热负荷自动化计算工具
---
## 📖 简介
本项目基于 Python 与 **CoolProp** 库，实现了 R134a 系统的快速物性检索。

## 🚀 核心功能
* **一键物性查询**：输入蒸发压力即可获得饱和气体焓值。
* **可视化分析**：自动生成压焓图 (P-h Diagram)。
